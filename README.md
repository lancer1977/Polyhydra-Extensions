# Core.Extensions

A .NET library providing common extension methods and utilities for PolyhydraGames projects.

## Namespace
All types are in the `PolyhydraGames.Extensions` namespace.

```csharp
using PolyhydraGames.Extensions;
```

## Build & Test

```bash
# Build the library
dotnet build Extensions/Extensions.csproj

# Run tests
dotnet test

# Package for NuGet
dotnet pack Extensions/Extensions.csproj -c Release
```
